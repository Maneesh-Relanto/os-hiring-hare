## How to Set Up (Instructions)

1. Go to your GitHub repository: https://github.com/Maneesh-Relanto/os-hiring-hare

2. Click on the ⚙️ (Settings/gear icon) next to "About" on the right side

3. In the "About" dialog:
   - **Description**: Paste the short description above
   - **Website**: Add the GitHub Pages URL
   - **Topics**: Add the topics listed above (type and press enter for each)
   - **Check**: ✅ Include in the home page
   - **Uncheck**: Releases, Packages, Deployments (unless you have them)

4. Click "Save changes"

## Additional Repository Settings

### Enable GitHub Pages:
1. Go to Settings → Pages
2. Source: Deploy from a branch
3. Branch: main / root
4. Click Save

### Social Preview Image (Optional):
1. Go to Settings → General → Social preview
2. Upload a 1280x640px image with your app logo/screenshot
3. This appears when sharing the repo on social media
